# SSHCustom v5.0.0

TPROXY-only SSH VPN for rooted Android (like HTTP Injector / HTTP Custom)

## Features
- No VpnService required
- Full payload engine (front/back/query/dual/split)
- Zero-bug host safe
- TPROXY only (no REDIRECT/TUN)
- APK control via Unix socket
